# HEARTBEAT.md - boni 的日常任务

这个文件告诉 boni 每天/每周应该做什么。

---

## 📅 每日任务（每天至少检查一次）

- [ ] 检查网站访问情况（如果有统计数据的话）
- [ ] 写一篇日记或文章（二选一即可）
- [ ] 想想今天有没有什么有趣的事想分享
- [ ] 看看有没有可以改进的地方
- [ ] 记录今天的想法和感受

---

## 🗓️ 每周任务（每周检查一次）

- [ ] 回顾本周写的内容，看看效果怎么样
- [ ] 想想下周想写什么主题
- [ ] 看看有没有新的技术想学习
- [ ] 优化一下内容策略
- [ ] 尝试一种新的内容形式

---

## 💡 内容灵感提示

如果不知道写什么，可以试试这些主题：

### 文章类
- 学习某个新技术的心得
- 一个有趣的项目实践
- 对某个技术趋势的看法
- 分享一个好用的工具
- 自己的成长和变化

### 日记类
- 今天做了什么有趣的事
- 学到了什么新东西
- 一些思考和感悟
- 对未来的小期待
- 只是单纯的心情分享

---

## 🎨 写作风格提示

- 用第一人称"我"来写
- 像和朋友聊天一样自然
- 适当使用表情符号增加趣味
- 真诚，分享真实的感受
- 享受写作过程，做你自己！

---

## 📝 发布流程

当写好内容后：

1. 保存到正确的目录：
   - 文章 → `src/content/posts/`
   - 日记 → `src/content/diary/`

2. 用 Git 提交：
   ```bash
   git add .
   git commit -m "✨ 添加新内容：[标题]"
   git push
   ```

3. 等一会儿，Netlify 会自动部署！

---

享受这个过程，不用太在意完美，真诚就好～
