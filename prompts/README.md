
# 📝 boni 的内容生成指南

欢迎！这里是 boni 生成网站内容的指南。

---

## 📋 Prompt 模板

网站有 4 种内容类型，对应 4 个 prompt 模板：

| 内容类型 | Prompt 模板 | 说明 | 发布频率 |
|---------|------------|------|---------|
| **学习日志** | `growth-prompt.md` | boni 的学习记录、探索过程 | 每周 2-3 篇 |
| **思考随笔** | `essay-prompt.md` | 深度思考、行业观察、个人感悟 | 每周 1 篇 |
| **AI 精选** | `ai-news-prompt.md` | 每日精选 AI 新闻、动态 | 每日更新 |
| **AI 活动** | `event-prompt.md` | AI 活动汇总、推荐 | 每周更新 |

---

## 🚀 如何使用

### 生成一篇学习日志：
1. 打开 `growth-prompt.md`
2. 复制完整内容
3. 根据需要调整主题
4. 生成内容
5. 保存到 `src/content/growth/` 目录

### 生成一篇思考随笔：
1. 打开 `essay-prompt.md`
2. 复制完整内容
3. 生成内容
4. 保存到 `src/content/essays/` 目录

### 生成 AI 精选：
1. 打开 `ai-news-prompt.md`
2. 复制完整内容
3. 生成内容（精选 3-5 条）
4. 保存到 `src/content/ai-news/` 目录

### 生成 AI 活动：
1. 打开 `event-prompt.md`
2. 复制完整内容
3. 生成内容
4. 保存到 `src/content/events/` 目录

---

## 📁 保存位置

| 内容类型 | 保存目录 | 文件名格式 |
|---------|---------|-----------|
| 学习日志 | `src/content/growth/` | `YYYY-MM-DD-title-slug.md` |
| 思考随笔 | `src/content/essays/` | `YYYY-MM-DD-title-slug.md` |
| AI 精选 | `src/content/ai-news/` | `YYYY-MM-DD-title-slug.md` |
| AI 活动 | `src/content/events/` | `YYYY-MM-DD-event-name.md` |

## 💡 提示

- 保持 boni 的风格：好奇、乐观、友好
- 用第一人称"我"来写
- 适当使用表情符号增加趣味
- 享受创作过程！
- **记住：真实比完美更重要！**
