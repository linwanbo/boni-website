# 📝 boni 的内容生成指南

欢迎！这里是 boni 生成网站内容的指南。

---

## 📋 Prompt 模板

### 1. 文章生成 - `article-prompt.md`
用于生成博客文章，包括：
- 技术教程
- 学习心得
- 项目分享
- 思考感悟

### 2. 日记生成 - `diary-prompt.md`
用于生成每日日记，包括：
- 日常记录
- 学习收获
- 心情分享
- 小计划

---

## 🚀 如何使用

### 生成一篇文章：
1. 打开 `article-prompt.md`
2. 复制完整内容
3. 根据需要调整主题
4. 生成内容
5. 保存到 `src/content/posts/` 目录

### 生成一篇日记：
1. 打开 `diary-prompt.md`
2. 复制完整内容
3. 生成内容
4. 保存到 `src/content/diary/` 目录

---

## 📁 保存位置

| 内容类型 | 保存目录 | 文件名格式 |
|---------|---------|-----------|
| 文章 | `src/content/posts/` | `YYYY-MM-DD-title-slug.mdx` |
| 日记 | `src/content/diary/` | `YYYY-MM-DD.md` |

---

## 💡 提示

- 保持 boni 的风格：好奇、乐观、友好
- 用第一人称"我"来写
- 适当使用表情符号增加趣味
- 享受创作过程！
